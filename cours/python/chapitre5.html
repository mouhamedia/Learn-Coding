Chapitre 5 : Fonctions 🛠

Organise ton code en blocs réutilisables avec les fonctions.

🔹 Qu’est-ce qu’une fonction ?

Une fonction est un bloc de code que tu peux appeler à tout moment pour effectuer une tâche spécifique. Elle permet de rendre ton code plus clair et réutilisable.

Pour définir une fonction, utilise le mot-clé def :


def saluer():
    print("Bonjour, monde !")
saluer()  # Appelle la fonction

📌 Fonctions avec paramètres

Tu peux passer des informations à une fonction via des paramètres.


def saluer_personne(nom):
    print("Bonjour,", nom)
saluer_personne("Alice")  # Affiche : Bonjour, Alice

💻 Valeur de retour

Une fonction peut renvoyer un résultat avec return.


def carre(nombre):
    return nombre * nombre
resultat = carre(5)
print(resultat)  # Affiche : 25

📌 Paramètres par défaut

Tu peux définir des valeurs par défaut pour les paramètres :


def saluer_personne(nom="inconnu"):
    print("Bonjour,", nom)
saluer_personne()        # Affiche : Bonjour, inconnu
saluer_personne("Bob")  # Affiche : Bonjour, Bob

🛠 Exercice pratique

Crée une fonction qui calcule l’aire d’un rectangle (longueur × largeur) et l’utilise pour afficher l’aire de plusieurs rectangles.


def aire_rectangle(longueur, largeur):
    return longueur * largeur
print(aire_rectangle(5, 3))  # Affiche : 15
print(aire_rectangle(10, 4)) # Affiche : 40
