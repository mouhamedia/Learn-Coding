Chapitre 4 : Boucles (for, while) 🔄

Répète des actions facilement avec les boucles en Python.

🔹 Qu’est-ce qu’une boucle ?

Les boucles permettent d’exécuter un bloc de code plusieurs fois. Python propose deux types de boucles : for et while.

📌 Boucle for

La boucle for est utilisée pour parcourir une séquence (comme une liste, une chaîne, ou une plage de nombres).


for i in range(5):  # De 0 à 4
    print(i)

Ce code affiche : 0 1 2 3 4. La fonction range(n) génère une séquence de nombres de 0 à n-1.

Exemple avec une liste :


fruits = ["pomme", "banane", "orange"]
for fruit in fruits:
    print("J’aime les", fruit)

Sortie : J’aime les pomme, J’aime les banane, J’aime les orange.

📌 Boucle while

La boucle while répète un bloc tant qu’une condition est vraie.


compteur = 1
while compteur <= 5:
    print(compteur)
    compteur += 1  # Incrémente le compteur

Ce code affiche : 1 2 3 4 5. Attention à toujours mettre à jour la condition pour éviter une boucle infinie !

💻 Contrôle des boucles

Tu peux modifier le flux d’une boucle avec :

  • break : Sort de la boucle immédiatement.
  • continue : Passe à l’itération suivante.

for i in range(10):
    if i == 3:
        continue  # Ignore 3
    if i == 7:
        break    # Arrête à 7
    print(i)

Sortie : 0 1 2 4 5 6.

🛠 Exercice pratique

Crée un programme qui demande un nombre à l’utilisateur et affiche la table de multiplication de ce nombre (de 1 à 10).


nombre = int(input("Entrez un nombre : "))
for i in range(1, 11):
    print(nombre, "x", i, "=", nombre * i)
