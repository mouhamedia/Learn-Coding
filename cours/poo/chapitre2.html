Chapitre 2 : Classes et objets 🧱

Apprends à créer des classes et des objets pour structurer ton code comme un pro !

🔹 Qu’est-ce qu’une classe ?

Une classe est comme un plan ou un modèle pour créer des objets. Elle définit :

  • Les attributs (données, comme le nom ou l’âge).
  • Les méthodes (actions que l’objet peut effectuer).

🔹 Qu’est-ce qu’un objet ?

Un objet est une instance d’une classe, c’est-à-dire un élément créé à partir de ce plan. Par exemple, une classe Personne peut créer des objets comme "Alice" ou "Bob".

💻 Exemple en Python

Voici une classe Personne avec un attribut (nom, âge) et une méthode (se présenter) :


class Personne:
    def __init__(self, nom, age):
        self.nom = nom
        self.age = age
    
    def se_presenter(self):
        print(f"Salut, je m'appelle {self.nom} et j'ai {self.age} ans.")

# Créer des objets
personne1 = Personne("Alice", 25)
personne2 = Personne("Bob", 30)

personne1.se_presenter()  # Affiche : Salut, je m'appelle Alice et j'ai 25 ans.
personne2.se_presenter()  # Affiche : Salut, je m'appelle Bob et j'ai 30 ans.

Dans cet exemple, Personne est la classe, et personne1 et personne2 sont des objets.

📌 Points clés

  • Une classe est un modèle, un objet est une instance de ce modèle.
  • Le constructeur __init__ initialise les attributs d’un objet.
  • Les méthodes sont des fonctions définies dans une classe.
