Chapitre 7 : Constructeurs et destructeurs 🏗️

Apprends à initialiser et nettoyer tes objets correctement !

🔹 Qu’est-ce qu’un constructeur ?

Un constructeur est une méthode spéciale (__init__ en Python) appelée lors de la création d’un objet pour initialiser ses attributs.

🔹 Qu’est-ce qu’un destructeur ?

Un destructeur (__del__ en Python) est appelé quand un objet est supprimé pour libérer des ressources.

💻 Exemple en Python

Voici une classe Livre avec un constructeur et un destructeur :


class Livre:
    def __init__(self, titre, auteur="Inconnu"):
        self.titre = titre
        self.auteur = auteur
        print(f"Livre {self.titre} créé.")
    
    def __del__(self):
        print(f"Livre {self.titre} supprimé.")

livre = Livre("Python 101")
del livre  # Affiche : Livre Python 101 supprimé.

📌 Points clés

  • Le constructeur __init__ initialise les attributs d’un objet.
  • Le destructeur __del__ est appelé automatiquement à la suppression.
  • Les paramètres par défaut (ex. : auteur="Inconnu") rendent le code flexible.
