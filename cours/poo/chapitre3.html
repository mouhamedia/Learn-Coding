Chapitre 3 : Attributs et méthodes 🛠️

Comprends les données et les actions d’une classe pour donner vie à tes objets !

🔹 Qu’est-ce qu’un attribut ?

Les attributs sont les données stockées dans un objet, comme le nom ou la couleur. Il existe deux types :

  • Attributs d’instance : Propres à chaque objet.
  • Attributs de classe : Partagés par tous les objets d’une classe.

🔹 Qu’est-ce qu’une méthode ?

Les méthodes sont des fonctions définies dans une classe qui décrivent ce que l’objet peut faire.

💻 Exemple en Python

Voici une classe Chien avec un attribut de classe (espece) et une méthode (aboyer) :


class Chien:
    espece = "Canis lupus"  # Attribut de classe
    
    def __init__(self, nom, race):
        self.nom = nom      # Attribut d’instance
        self.race = race
    
    def aboyer(self):
        print(f"{self.nom} dit : Wouf !")
    
    def info(self):
        print(f"{self.nom} est un {self.race} de l'espèce {self.espece}")

chien1 = Chien("Rex", "Berger")
chien1.aboyer()  # Affiche : Rex dit : Wouf !
chien1.info()    # Affiche : Rex est un Berger de l'espèce Canis lupus
print(Chien.espece)  # Affiche : Canis lupus

📌 Points clés

  • Les attributs d’instance sont définis dans __init__.
  • Les attributs de classe sont partagés par tous les objets.
  • Les méthodes utilisent self pour accéder aux attributs de l’objet.
