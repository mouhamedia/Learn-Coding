Chapitre 6 : Polymorphisme 🎭

Fais en sorte que des objets différents réagissent différemment à la même méthode !

🔹 Qu’est-ce que le polymorphisme ?

Le polymorphisme permet à des objets de différentes classes de répondre à la même méthode de manière différente. Cela rend le code flexible.

🔹 Comment ça marche ?

Une classe fille peut redéfinir une méthode héritée pour adapter son comportement.

💻 Exemple en Python

Voici une classe Animal avec des classes filles Chat et Chien :


class Animal:
    def parler(self):
        pass

class Chat(Animal):
    def parler(self):
        print("Miaou !")

class Chien(Animal):
    def parler(self):
        print("Wouf !")

animaux = [Chat(), Chien()]
for animal in animaux:
    animal.parler()  # Affiche : Miaou ! puis Wouf !

📌 Points clés

  • Le polymorphisme permet à une méthode d’avoir des comportements différents.
  • Les classes filles redéfinissent les méthodes du parent.
  • Utilisé pour écrire du code flexible et réutilisable.
