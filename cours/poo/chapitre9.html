Chapitre 9 : Relations entre classes 🤝

Apprends comment les classes peuvent interagir entre elles !

🔹 Types de relations

Les classes peuvent être liées par :

  • Association : Une classe utilise une autre (ex. : une classe Ecole utilise Etudiant).
  • Agrégation : Une classe contient d’autres objets, mais ils peuvent exister indépendamment.
  • Composition : Une classe possède des objets qui n’existent pas sans elle.

💻 Exemple en Python

Voici une classe Ecole en relation d’agrégation avec Etudiant :


class Etudiant:
    def __init__(self, nom):
        self.nom = nom
    
    def info(self):
        print(f"Etudiant : {self.nom}")

class Ecole:
    def __init__(self, nom):
        self.nom = nom
        self.etudiants = []
    
    def ajouter_etudiant(self, etudiant):
        self.etudiants.append(etudiant)
    
    def afficher_etudiants(self):
        for etudiant in self.etudiants:
            etudiant.info()

ecole = Ecole("Lycée")
etudiant1 = Etudiant("Alice")
etudiant2 = Etudiant("Bob")
ecole.ajouter_etudiant(etudiant1)
ecole.ajouter_etudiant(etudiant2)
ecole.afficher_etudiants()  # Affiche : Etudiant : Alice, Etudiant : Bob

📌 Points clés

  • L’association est une relation simple entre classes.
  • L’agrégation implique une relation "a-un" sans dépendance forte.
  • La composition lie la durée de vie des objets.
