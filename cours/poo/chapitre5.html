Chapitre 5 : Héritage 🌳

Réutilise du code en faisant hériter des classes les unes des autres !

🔹 Qu’est-ce que l’héritage ?

L’héritage permet à une classe (classe fille) de récupérer les attributs et méthodes d’une autre classe (classe parent). Cela évite de réécrire du code.

🔹 Comment ça marche ?

Une classe fille est définie avec class Fille(Parent):. On utilise super() pour appeler le constructeur du parent.

💻 Exemple en Python

Voici une classe Vehicule (parent) et Voiture (fille) :


class Vehicule:
    def __init__(self, marque):
        self.marque = marque
    
    def klaxonner(self):
        print("Tut tut !")

class Voiture(Vehicule):
    def __init__(self, marque, couleur):
        super().__init__(marque)
        self.couleur = couleur
    
    def description(self):
        print(f"C'est une {self.couleur} {self.marque}")

ma_voiture = Voiture("Toyota", "bleue")
ma_voiture.klaxonner()    # Affiche : Tut tut !
ma_voiture.description()  # Affiche : C'est une bleue Toyota

📌 Points clés

  • L’héritage permet de réutiliser le code d’une classe parent.
  • super() appelle les méthodes ou le constructeur du parent.
  • Une classe fille peut ajouter ses propres attributs/méthodes.
