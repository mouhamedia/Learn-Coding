Chapitre 2 : Installation et configuration 🛠️

Apprends à configurer ton environnement de développement pour créer des applications React performantes.

🔹 Pourquoi configurer un environnement React ?

Pour développer avec React, tu as besoin d’un environnement avec Node.js, un gestionnaire de paquets (npm), et des outils comme create-react-app. Une configuration correcte garantit un développement fluide et des tests efficaces.

Outils nécessaires :

  • Node.js : Exécute JavaScript côté serveur et gère les dépendances.
  • npm : Installe les bibliothèques React et autres paquets.
  • create-react-app : Crée une structure de projet préconfigurée.
  • Éditeur de code : VS Code avec extensions comme ESLint et Prettier.

🔹 Exemple 1 : Créer un projet React

Utilise create-react-app pour démarrer un projet :

npx create-react-app mon-app
cd mon-app
npm start

Explication :

  • npx : Exécute create-react-app sans installation globale.
  • mon-app : Nom du projet.
  • npm start : Lance le serveur de développement à http://localhost:3000.

Résultat : Ouvre http://localhost:3000 pour voir l’application React par défaut.

🔹 Exemple 2 : Structure du projet

Voici la structure générée par create-react-app :

mon-app/
├── node_modules/           # Dépendances
├── public/                 # Fichiers statiques
│   ├── index.html          # Page HTML principale
│   ├── favicon.ico         # Icône du site
├── src/                    # Code source
│   ├── App.js              # Composant principal
│   ├── index.js            # Point d’entrée
├── package.json            # Configuration et dépendances
├── README.md               # Documentation

Résultat : src/App.js contient le composant principal, rendu dans public/index.html.

🔹 Exemple 3 : Personnaliser index.html

Modifie public/index.html pour changer le titre :

<!-- public/index.html -->
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="utf-8">
  <title>Ma Première App React</title>
</head>
<body>
  <div id="root"></div>
</body>
</html>

Résultat : Le titre de la page devient “Ma Première App React”.

🔹 Exemple 4 : Installer une bibliothèque

Ajoute react-icons pour utiliser des icônes :

npm install react-icons

Utilise une icône dans src/App.js :

// src/App.js
import { FaHeart } from 'react-icons/fa';

function App() {
  return (
    <div>
      <h1>Mon App</h1>
      <FaHeart />
    </div>
  );
}

export default App;

Résultat : Affiche un cœur à côté de “Mon App”.

🔹 Exemple 5 : Configurer ESLint

Installe ESLint pour vérifier le code :

npm init @eslint/config

Suis les invites pour configurer React et JSX. Résultat : ESLint signale les erreurs de code dans VS Code.

🔹 Erreurs courantes à éviter

  • Node.js absent : Installe Node.js depuis nodejs.org.
  • Mauvais répertoire : Exécute npm start dans le dossier du projet.
  • Port occupé : Si localhost:3000 est utilisé, change le port avec PORT=3001 npm start.
