Chapitre 2 : Installation et premier script

Apprends à installer Node.js et à écrire ton premier script.

Installer Node.js

Pour utiliser Node.js, tu dois l’installer sur ton système. Télécharge la version LTS (Long Term Support) depuis nodejs.org.

Étapes d’installation (Windows/Mac/Linux) :

  1. Télécharge l’installateur pour ton système.
  2. Suis les instructions de l’assistant d’installation.
  3. Vérifie l’installation en ouvrant un terminal et en exécutant :
node --version
npm --version

Ces commandes affichent les versions de Node.js et npm (le gestionnaire de paquets).

Écrire ton premier script

Crée un fichier app.js avec un code simple :

// app.js
const greeting = 'Bienvenue dans Node.js !';
console.log(greeting);

Exécute le script avec :

node app.js

Sortie : Bienvenue dans Node.js !

Exemple pratique : Calculatrice simple

Voici un script Node.js qui effectue une addition à partir des arguments de la ligne de commande :

// calculator.js
const args = process.argv.slice(2);
const num1 = Number(args[0]);
const num2 = Number(args[1]);
if (isNaN(num1) || isNaN(num2)) {
  console.log('Veuillez fournir deux nombres valides.');
} else {
  const sum = num1 + num2;
  console.log(`La somme de ${num1} et ${num2} est : ${sum}`);
}

Exécute avec :

node calculator.js 5 3

Sortie : La somme de 5 et 3 est : 8

Le tableau process.argv contient les arguments passés à la commande, avec les deux premiers éléments étant le chemin de Node.js et du script.

Bonnes pratiques :

  • Utilise la version LTS de Node.js pour la stabilité.
  • Teste tes scripts dans un terminal pour repérer rapidement les erreurs.
  • Valide les entrées utilisateur (comme dans l’exemple ci-dessus) pour éviter les erreurs.
