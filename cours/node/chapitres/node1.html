Chapitre 1 : Introduction à Node.js

Découvre les bases de Node.js, une plateforme pour exécuter JavaScript côté serveur.

Qu’est-ce que Node.js ?

Node.js est une plateforme open-source qui permet d’exécuter du JavaScript en dehors du navigateur, en utilisant le moteur V8 de Google Chrome. Il est idéal pour créer des applications serveur rapides et évolutives.

Pourquoi utiliser Node.js ?

Node.js est populaire pour :

  • Performance : Son modèle non bloquant et événementiel le rend rapide.
  • Écosystème : Un vaste écosystème de modules via npm.
  • JavaScript partout : Utilise le même langage pour le client et le serveur.
  • Communauté : Une large communauté et des frameworks comme Express.js.

Comment fonctionne Node.js ?

Node.js utilise un modèle asynchrone basé sur des événements. Les opérations d’entrée/sortie (comme la lecture de fichiers ou les requêtes réseau) sont non bloquantes, permettant de gérer plusieurs tâches simultanément.

console.log('Début');
setTimeout(() => {
  console.log('Tâche asynchrone');
}, 1000);
console.log('Fin');

Sortie :

Début
Fin
Tâche asynchrone

Dans cet exemple, setTimeout est asynchrone, permettant à Fin de s’afficher avant la tâche différée.

Exemple pratique : Hello World

Voici un exemple simple de script Node.js :

// hello.js
console.log('Hello, Node.js !');

Pour exécuter ce script, enregistre-le dans un fichier hello.js et utilise la commande :

node hello.js

Sortie : Hello, Node.js !

Bonnes pratiques :

  • Assure-toi d’avoir une version récente de Node.js (au moins 18.x) pour bénéficier des dernières fonctionnalités.
  • Comprends le modèle asynchrone pour éviter les erreurs courantes.
  • Utilise un éditeur comme VS Code avec des extensions Node.js pour une meilleure expérience.
