Chapitre 7 : Express.js – Introduction

Découvre Express.js, un framework pour créer des serveurs web rapidement.

Qu’est-ce qu’Express.js ?

Express.js est un framework minimaliste pour Node.js qui simplifie la création de serveurs web et d’APIs, avec un système de routing et de middleware robuste.

Installer Express.js

Crée un projet et installe Express :

npm init -y
npm install express

Premier serveur Express

Voici un serveur Express de base :

const express = require('express');
const app = express();
const port = 3000;

app.get('/', (req, res) => {
  res.send('Bienvenue sur Express !');
});

app.listen(port, () => {
  console.log(`Serveur démarré sur http://localhost:${port}`);
});

Exécute avec node app.js et ouvre http://localhost:3000.

Exemple pratique : API utilisateurs

Voici une API simple pour gérer une liste d’utilisateurs :

const express = require('express');
const app = express();
const port = 3000;

const users = [
  { id: 1, name: 'Jean' },
  { id: 2, name: 'Marie' }
];

app.get('/api/users', (req, res) => {
  res.json(users);
});

app.get('/api/users/:id', (req, res) => {
  const user = users.find(u => u.id === parseInt(req.params.id));
  if (!user) {
    res.status(404).json({ error: 'Utilisateur non trouvé' });
  } else {
    res.json(user);
  }
});

app.listen(port, () => {
  console.log(`Serveur démarré sur http://localhost:${port}`);
});

Teste avec :

  • curl http://localhost:3000/api/users pour tous les utilisateurs.
  • curl http://localhost:3000/api/users/1 pour un utilisateur spécifique.

Sortie pour /api/users :

[{"id":1,"name":"Jean"},{"id":2,"name":"Marie"}]

Bonnes pratiques :

  • Utilise res.json pour envoyer des réponses JSON.
  • Gère les erreurs (comme les 404) pour améliorer l’expérience utilisateur.
  • Organise ton code en modules pour les projets plus complexes.
