Chapitre 7 : Les formulaires 📝

Crée des formulaires interactifs pour collecter des données !

🔹 Qu’est-ce qu’un formulaire ?

Un formulaire permet aux utilisateurs d’entrer des données (texte, choix, etc.) et de les envoyer à un serveur. Il utilise la balise <form>.

Analogie : Un formulaire est comme un questionnaire papier où l’utilisateur remplit des champs.

🔹 Structure d’un formulaire

  • <form> : Contient le formulaire, avec les attributs action (destination des données) et method (GET ou POST).
  • <label> : Associe un texte à un champ pour l’accessibilité.
  • <input> : Champs de saisie (texte, bouton radio, etc.).
  • <button> : Bouton pour soumettre le formulaire.

🔹 Types d’entrées

  • type="text" : Champ de texte.
  • type="radio" : Bouton radio pour un choix unique.
  • type="checkbox" : Case à cocher pour plusieurs choix.
  • type="submit" : Bouton d’envoi.

💻 Exemple pratique

Voici un formulaire simple :


<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Formulaire HTML</title>
</head>
<body>
  <h1>Formulaire de contact</h1>
  <form action="/submit" method="post">
    <label for="nom">Nom :</label>
    <input type="text" id="nom" name="nom">
    <button type="submit">Envoyer</button>
  </form>
</body>
</html>

Cet exemple affiche un formulaire avec un champ texte et un bouton d’envoi.

📌 Points clés

  • Les formulaires collectent des données avec <form>.
  • Associez <label> à <input> pour l’accessibilité.
  • Utilisez différents types d’<input> selon les besoins.

🚀 Pour aller plus loin

Ajoute un champ type="email" ou des boutons radio pour choisir une option.
