Chapitre 2 : Structure d’un document HTML 🏗️

Comprends comment organiser une page HTML correctement !

🔹 La structure de base

Un document HTML suit une structure standard avec des balises principales :

  • <!DOCTYPE html> : Indique que c’est un document HTML5.
  • <html> : Contient tout le contenu de la page.
  • <head> : Contient les métadonnées (titre, encodage, etc.).
  • <body> : Contient le contenu visible (textes, images, etc.).

Analogie : Une page HTML est comme un livre : <head> est la page de titre, et <body> est le contenu des chapitres.

🔹 Les métadonnées

Dans <head>, on trouve :

  • <title> : Le titre affiché dans l’onglet du navigateur.
  • <meta charset="UTF-8"> : Définit l’encodage des caractères.
  • <meta name="viewport"> : Rend la page responsive.

💻 Exemple pratique

Voici un document HTML structuré :


<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Mon site</title>
</head>
<body>
  <h1>Bienvenue sur mon site</h1>
  <p>Ce site est en construction !</p>
</body>
</html>

Cet exemple crée une page avec un titre et un paragraphe.

📌 Points clés

  • Un document HTML a une structure fixe : DOCTYPE, html, head, body.
  • Le <head> contient les informations non visibles.
  • Le <body> contient tout ce que l’utilisateur voit.

🚀 Pour aller plus loin

Ajoute une balise <meta name="description"> pour décrire ton site, ou modifie le titre pour inclure ton nom.
