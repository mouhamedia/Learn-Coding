Chapitre 4 : Liens et images 🔗🖼️

Connecte des pages et affiche des images pour enrichir ton site !

🔹 Les hyperliens

Les liens hypertextes permettent de naviguer entre des pages web en utilisant la balise <a> avec l’attribut href pour indiquer la destination.

Analogie : Un lien est comme une porte qui mène à une autre pièce (une page web).

Attributs utiles :

  • href : URL ou chemin vers la page cible.
  • target="_blank" : Ouvre le lien dans un nouvel onglet.

🔹 Les images

Les images sont ajoutées avec la balise <img>, qui utilise src pour le chemin de l’image et alt pour une description textuelle (important pour l’accessibilité).

Attributs utiles :

  • src : Chemin ou URL de l’image.
  • alt : Texte alternatif pour les lecteurs d’écran.
  • width ou height : Taille de l’image en pixels.

💻 Exemple pratique

Voici une page avec un lien et une image :


<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Liens et images</title>
</head>
<body>
  <h1>Ma page</h1>
  <p>Visitez <a href="https://example.com" target="_blank">Example.com</a> pour en savoir plus.</p>
  <img src="https://via.placeholder.com/150" alt="Image de placeholder" width="150">
</body>
</html>

Cet exemple affiche un lien cliquable et une image.

📌 Points clés

  • Les liens relient les pages avec <a href>.
  • Les images nécessitent un chemin valide et un texte alt.
  • Utilisez des chemins relatifs pour les fichiers locaux, absolus pour les URLs.

🚀 Pour aller plus loin

Crée un lien vers une autre page de ton site (ex. : chapitre5.html) ou ajoute plusieurs images avec des tailles différentes.
