Chapitre 5 : Les listes HTML 📋

Organise ton contenu avec des listes claires et structurées !

🔹 Pourquoi utiliser des listes ?

Les listes permettent d’organiser des informations de manière claire, comme une liste de courses ou des étapes numérotées.

Analogie : Une liste HTML est comme un menu au restaurant : soit une simple liste d’options (non ordonnée), soit des étapes dans l’ordre (ordonnée).

🔹 Types de listes

  • Liste non ordonnée (<ul>) : Utilise des puces pour des éléments sans ordre spécifique.
  • Liste ordonnée (<ol>) : Utilise des numéros ou lettres pour des éléments dans un ordre précis.
  • Liste de définition (<dl>) : Associe des termes à leurs définitions.

💻 Exemple pratique

Voici un exemple avec différents types de listes :


<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Listes HTML</title>
</head>
<body>
  <h1>Mes listes</h1>
  <h2>Liste de courses</h2>
  <ul>
    <li>Pommes</li>
    <li>Bananes</li>
  </ul>
  <h2>Étapes pour cuisiner</h2>
  <ol>
    <li>Préchauffer le four</li>
    <li>Mélanger les ingrédients</li>
  </ol>
</body>
</html>

Cet exemple affiche une liste non ordonnée et une liste ordonnée.

📌 Points clés

  • <ul> pour les listes à puces, <ol> pour les listes numérotées.
  • Chaque élément de liste est défini par <li>.
  • Les listes rendent le contenu facile à lire et structuré.
