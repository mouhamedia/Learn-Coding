Chapitre 9 : Intégration CSS et JS 🎨🖱️

Ajoute du style et de l’interactivité à tes pages web !

🔹 Qu’est-ce que CSS et JavaScript ?

CSS (Cascading Style Sheets) stylise les pages HTML (couleurs, tailles, etc.), tandis que JavaScript ajoute de l’interactivité (boutons cliquables, animations).

Analogie : HTML est le squelette, CSS est la peinture, et JavaScript est le moteur qui anime la page.

🔹 Intégrer CSS

Tu peux ajouter CSS via :

  • <style> dans <head>.
  • <link rel="style.css" href="style.css"> pour un fichier externe.
  • Exemple : style="color: blue;" directement dans une balise.

🔹 Intégrer

Tu peux ajouter JavaScript via :

  • <script> dans <head> ou <body>.
  • <script src="script.js">< pour un fichier externe.
  • Exemple : onclick="maFonction()" dans une balise.

💻 Exemple pratique

Voici une page avec CSS et JavaScript :


<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>CSS et JS</title>
  <style>
    h1 { color: blue; }
  </style>
</head>
<body>
  <h1>Page stylée</h1>
  <button onclick="montrerMessage">Cliquez ici !</button>
  <script>
    function montrerMessage() {
      alert('Bonjour !');
    }
  </script>
</body>
</html>

Cet exemple affiche un titre bleu et un bouton qui montre une alerte.

📌 Points clés

  • CSS stylise les éléments HTML.
  • JavaScript ajoute de l’interactivité.
  • Liez les fichiers externes pour organiser le code.
