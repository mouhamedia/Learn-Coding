Chapitre 1 : Introduction à HTML 📄

Le premier pas pour créer des pages web !

🔹 Qu’est-ce que HTML ?

HTML (HyperText Markup Language) est le langage utilisé pour structurer le contenu des pages web. C’est comme les fondations d’une maison : il définit où vont les murs, les portes, et les fenêtres.

Analogie : Imagine HTML comme un plan d’architecte qui indique où placer le texte, les images, et les boutons sur une page web.

🔹 À quoi ça sert ?

HTML permet de :

  • Créer des titres, paragraphes, et listes.
  • Ajouter des images, liens, et formulaires.
  • Structurer une page pour qu’elle soit lisible par les navigateurs.

🔹 Comment ça fonctionne ?

HTML utilise des balises (tags) entourées de < et >. Chaque balise a un rôle, comme <p> pour un paragraphe.

💻 Exemple pratique

Voici une page HTML simple :


<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Ma première page</title>
</head>
<body>
  <h1>Bienvenue !</h1>
  <p>Ceci est ma première page web.</p>
</body>
</html>

Cet exemple affiche un titre et un paragraphe dans le navigateur.

📌 Points clés

  • HTML est le squelette d’une page web.
  • Les balises définissent la structure du contenu.
  • Un fichier HTML commence toujours par <!DOCTYPE html>.

🚀 Pour aller plus loin

Modifie l’exemple pour ajouter un deuxième paragraphe ou change le texte du titre.
