Chapitre 7 : Tableaux ☕

Stocke plusieurs valeurs dans une seule variable.

🔹 Qu’est-ce qu’un tableau ?

Un tableau est une liste de valeurs du même type, comme une rangée de boîtes numérotées.


public class Tableaux {
    public static void main(String[] args) {
        int[] nombres = {1, 2, 3, 4, 5};
        System.out.println("Premier élément : " + nombres[0]);
    }
}

📌 Parcourir un tableau

Utilise une boucle pour accéder à chaque élément.


public class ParcoursTableau {
    public static void main(String[] args) {
        int[] notes = {12, 15, 18};
        for (int i = 0; i < notes.length; i++) {
            System.out.println("Note " + i + " : " + notes[i]);
        }
    }
}

💻 Boucle for-each

Une manière plus simple de parcourir un tableau :


public class ForEach {
    public static void main(String[] args) {
        String[] fruits = {"Pomme", "Banane", "Orange"};
        for (String fruit : fruits) {
            System.out.println("Fruit : " + fruit);
        }
    }
}

🛠 Exercice pratique

Crée un programme qui calcule la somme des éléments d’un tableau de nombres.


public class SommeTableau {
    public static void main(String[] args) {
        int[] nombres = {10, 20, 30, 40};
        int somme = 0;
        for (int nombre : nombres) {
            somme += nombre;
        }
        System.out.println("Somme : " + somme);
    }
}
