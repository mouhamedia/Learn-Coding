Chapitre 9 : Encapsulation ☕

Protège les données de tes objets avec l’encapsulation.

🔹 Qu’est-ce que l’encapsulation ?

L’encapsulation consiste à cacher les données d’un objet et à n’y accéder qu’à travers des méthodes spéciales (getters et setters).

📌 Attributs privés

Utilise private pour limiter l’accès direct aux attributs.


public class Compte {
    private double solde;

    public Compte(double s) {
        solde = s;
    }
}

💻 Getters et setters

Les getters récupèrent la valeur, les setters la modifient.


public class Compte {
    private double solde;

    public Compte(double s) {
        solde = s;
    }

    public double getSolde() {
        return solde;
    }

    public void setSolde(double s) {
        if (s >= 0) {
            solde = s;
        }
    }

    public static void main(String[] args) {
        Compte compte = new Compte(100.0);
        compte.setSolde(150.0);
        System.out.println("Solde : " + compte.getSolde());
    }
}

🛠 Exercice pratique

Crée une classe Etudiant avec un attribut privé note et des getters/setters pour y accéder.


public class Etudiant {
    private int note;

    public Etudiant(int n) {
        note = n;
    }

    public int getNote() {
        return note;
    }

    public void setNote(int n) {
        if (n >= 0 && n <= 20) {
            note = n;
        }
    }

    public static void main(String[] args) {
        Etudiant e = new Etudiant(15);
        e.setNote(18);
        System.out.println("Note : " + e.getNote());
    }
}
