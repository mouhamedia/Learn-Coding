Chapitre 10 : Gestion des utilisateurs et permissions 🔐

Apprends à sécuriser une base de données en gérant les utilisateurs et leurs permissions.

🔹 Créer un utilisateur

Utilise CREATE USER pour ajouter un utilisateur (syntaxe MySQL).


CREATE USER 'utilisateur'@'localhost' IDENTIFIED BY 'motdepasse';

Résultat : Crée un utilisateur nommé "utilisateur" avec le mot de passe "motdepasse".

🔹 Accorder des permissions

Utilise GRANT pour donner des autorisations.


GRANT SELECT, INSERT ON Ecole.Etudiants TO 'utilisateur'@'localhost';

Résultat : L'utilisateur peut lire et insérer des données dans la table Etudiants.

🔹 Révoquer des permissions

Utilise REVOKE pour retirer des autorisations.


REVOKE INSERT ON Ecole.Etudiants FROM 'utilisateur'@'localhost';

Résultat : Retire la permission d'insérer des données.

📝 Exercice

Crée un utilisateur nommé "lecteur" avec un mot de passe, et donne-lui uniquement la permission de lire (SELECT) la table "Livres".
