Chapitre 2 : Premiers pas avec Vue.js

Apprends à créer une application simple avec Vue.js et découvre la réactivité.

La réactivité dans Vue.js

Vue.js est basé sur un système de réactivité : lorsque les données changent, l’interface se met à jour automatiquement. Les données sont définies dans l’option data d’un composant.

Créer une application simple

Voici un exemple d’application simple avec un compteur interactif :

<template>
  <div>
    <p>Compteur : {{ count }}</p>
    <button @click="increment">Incrémenter</button>
  </div>
</template>
<script>
export default {
  data() {
    return {
      count: 0
    };
  },
  methods: {
    increment() {
      this.count++;
    }
  }
};
</script>
<style scoped>
button {
  padding: 0.5rem 1rem;
  background-color: #42b883;
  color: white;
  border: none;
  border-radius: 4px;
}
button:hover {
  background-color: #2a9a78;
}
</style>

Dans cet exemple, cliquer sur le bouton incrémente count, et l’interface se met à jour automatiquement grâce à la réactivité.

Liaison de données

Vue.js utilise la directive v-bind pour lier des données aux attributs HTML et {{ }} pour afficher des données dans le texte.

<template>
  <div>
    <p>Message : {{ message }}</p>
    <input :value="message" @input="updateMessage">
  </div>
</template>
<script>
export default {
  data() {
    return {
      message: 'Bonjour Vue.js !'
    };
  },
  methods: {
    updateMessage(event) {
      this.message = event.target.value;
    }
  }
};
</script>
<style scoped>
input {
  padding: 0.5rem;
  margin-bottom: 1rem;
}
</style>

Ici, :value lie la valeur de l’input à message, et @input met à jour la donnée.

Exemple pratique : Mini formulaire

Voici un exemple combinant réactivité et liaison de données pour un formulaire simple :

<template>
  <div>
    <input :value="name" @input="updateName" placeholder="Entrez votre nom">
    <p v-if="name">Bonjour, {{ name }} !</p>
    <p v-else>Veuillez entrer un nom.</p>
  </div>
</template>
<script>
export default {
  data() {
    return {
      name: ''
    };
  },
  methods: {
    updateName(event) {
      this.name = event.target.value;
    }
  }
};
</script>
<style scoped>
input {
  padding: 0.5rem;
  margin-bottom: 1rem;
}
p {
  color: #333;
}
</style>

Ce formulaire affiche un message personnalisé en fonction du nom saisi, avec une condition pour gérer le cas où le champ est vide.

Bonnes pratiques :

  • Utilise des noms clairs pour les données et méthodes.
  • Garde les composants simples et focalisés sur une seule tâche.
  • Teste ton application dans un environnement local avec Vite pour une meilleure expérience de développement.
