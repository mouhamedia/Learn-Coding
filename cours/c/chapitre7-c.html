Chapitre 7 : Chaînes de caractères 🔤

Manipule du texte avec les chaînes de caractères en C.

🔹 Qu’est-ce qu’une chaîne ?

En C, une chaîne est un tableau de caractères terminé par \0 (caractère nul).


#include 

int main() {
    char message[] = "Bonjour";
    printf("%s\n", message);
    return 0;
}

📌 Bibliothèque string.h

Utilise des fonctions comme strlen, strcpy, strcmp.


#include 
#include 

int main() {
    char texte[] = "Bonjour";
    printf("Longueur : %zu\n", strlen(texte));
    char copie[50];
    strcpy(copie, texte);
    printf("Copie : %s\n", copie);
    return 0;
}

💻 Saisie de chaînes

Utilise fgets pour une saisie sécurisée (évite gets).


#include 

int main() {
    char nom[50];
    printf("Entrez votre nom : ");
    fgets(nom, 50, stdin);
    nom[strcspn(nom, "\n")] = 0; // Supprime \n
    printf("Salut, %s !\n", nom);
    return 0;
}
