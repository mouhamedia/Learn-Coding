Chapitre 2 : Variables et types de données 📚

Apprends à stocker et manipuler des données avec les variables en C.

🔹 Qu’est-ce qu’une variable ?

Une variable est un espace en mémoire pour stocker une valeur. En C, tu dois déclarer le type de la variable avant de l’utiliser.


#include 

int main() {
    int age = 25;
    float prix = 19.99;
    printf("Âge : %d, Prix : %.2f\n", age, prix);
    return 0;
}

%d est pour les entiers, %.2f pour les flottants avec 2 décimales.

📌 Types de données

Les principaux types en C sont :

  • int : Entier (ex. : 42, -10).
  • float : Flottant à simple précision (ex. : 3.14).
  • double : Flottant à double précision (plus précis).
  • char : Caractère (ex. : 'A').

💻 Entrée et sortie

Utilise scanf pour lire l’entrée utilisateur et printf pour afficher.


#include 

int main() {
    int age;
    printf("Entrez votre âge : ");
    scanf("%d", &age);
    printf("Vous avez %d ans.\n", age);
    return 0;
}

Note : Le & devant age indique l’adresse mémoire de la variable.

🛠 Exercice pratique

Crée un programme qui demande le nom et l’âge de l’utilisateur, puis affiche un message comme : Bonjour Alice, vous avez 25 ans.


#include 

int main() {
    char nom[50];
    int age;
    printf("Entrez votre nom : ");
    scanf("%s", nom);
    printf("Entrez votre âge : ");
    scanf("%d", &age);
    printf("Bonjour %s, vous avez %d ans.\n", nom, age);
    return 0;
}
