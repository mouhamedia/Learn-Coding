Chapitre 5 : Tableaux et pointeurs 📋

Manipule des collections de données et comprends les pointeurs.

🔹 Les tableaux

Un tableau est une collection d’éléments du même type, stockés consécutivement en mémoire.


#include 

int main() {
    int nombres[5] = {1, 2, 3, 4, 5};
    printf("Premier élément : %d\n", nombres[0]);
    return 0;
}

📌 Les pointeurs

Un pointeur est une variable qui stocke l’adresse mémoire d’une autre variable.


#include 

int main() {
    int x = 10;
    int *pointeur = &x;
    printf("Valeur de x : %d\n", *pointeur);
    printf("Adresse de x : %p\n", pointeur);
    return 0;
}

&x donne l’adresse, *pointeur accède à la valeur.

💻 Tableaux et pointeurs

Les tableaux sont étroitement liés aux pointeurs : le nom d’un tableau est un pointeur vers son premier élément.


#include 

int main() {
    int nombres[3] = {10, 20, 30};
    printf("Premier élément : %d\n", *nombres);
    printf("Deuxième élément : %d\n", *(nombres + 1));
    return 0;
}

🛠 Exercice pratique

Crée un programme qui stocke 5 nombres dans un tableau, calcule leur somme, et utilise un pointeur pour y accéder.


#include 

int main() {
    int nombres[5] = {1, 2, 3, 4, 5};
    int *ptr = nombres;
    int somme = 0;
    for (int i = 0; i < 5; i++) {
        somme += *(ptr + i);
    }
    printf("Somme : %d\n", somme);
    return 0;
}
