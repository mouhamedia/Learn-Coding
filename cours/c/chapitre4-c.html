Chapitre 4 : Fonctions en C 🛠

Organise ton code avec des fonctions réutilisables.

🔹 Qu’est-ce qu’une fonction ?

Une fonction est un bloc de code qui effectue une tâche spécifique et peut être appelé plusieurs fois.


#include 

void saluer() {
    printf("Bonjour, monde !\n");
}

int main() {
    saluer();
    return 0;
}

📌 Fonctions avec paramètres

Les paramètres permettent de passer des données à une fonction.


#include 

void saluer_personne(char *nom) {
    printf("Bonjour, %s !\n", nom);
}

int main() {
    saluer_personne("Alice");
    return 0;
}

💻 Valeur de retour

Une fonction peut renvoyer une valeur avec return.


#include 

int carre(int n) {
    return n * n;
}

int main() {
    int resultat = carre(5);
    printf("Carré de 5 : %d\n", resultat);
    return 0;
}

🛠 Exercice pratique

Crée une fonction qui calcule l’aire d’un rectangle (longueur × largeur) et affiche le résultat.


#include 

int aire_rectangle(int longueur, int largeur) {
    return longueur * largeur;
}

int main() {
    int l = 5, lrg = 3;
    printf("Aire : %d\n", aire_rectangle(l, lrg));
    return 0;
}
