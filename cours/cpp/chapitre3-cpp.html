Chapitre 3 : Variables et types de données ⚙️

Apprends à stocker des informations dans ton programme.

🔹 Qu’est-ce qu’une variable ?

Une variable est comme une boîte où tu peux mettre une valeur, comme un nombre ou du texte. En C++, tu dois dire quel type de valeur elle contiendra.

Exemple :


#include 
using namespace std;

int main() {
    int age = 25;
    cout << "J'ai " << age << " ans." << endl;
    return 0;
}

Ici, age est une boîte qui contient le nombre 25.

📌 Types de données

Les principaux types en C++ sont :

  • int : Nombres entiers (ex. : 5, -10).
  • float : Nombres décimaux (ex. : 3.14).
  • char : Caractères (ex. : 'A').
  • bool : Vrai ou faux (true, false).
  • string : Texte (ex. : "Bonjour") – nécessite #include .

💻 Constantes

Une constante est une variable dont la valeur ne change pas. Utilise const.


#include 
using namespace std;

int main() {
    const double PI = 3.14159;
    float rayon = 5.0;
    float aire = PI * rayon * rayon;
    cout << "Aire du cercle : " << aire << endl;
    return 0;
}

Ici, PI est une constante qui reste toujours 3.14159.
