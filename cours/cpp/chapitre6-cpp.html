Chapitre 6 : Fonctions ⚙️

Organise ton code avec des fonctions réutilisables.

🔹 Qu’est-ce qu’une fonction ?

Une fonction est comme une recette : un ensemble d’instructions que tu peux utiliser plusieurs fois.


#include 
using namespace std;

void saluer() {
    cout << "Bonjour, tout le monde !" << endl;
}

int main() {
    saluer();
    return 0;
}

📌 Fonctions avec paramètres

Tu peux envoyer des valeurs à une fonction pour qu’elle les utilise.


#include 
using namespace std;

void saluerPersonne(string nom) {
    cout << "Salut, " << nom << " !" << endl;
}

int main() {
    saluerPersonne("Alice");
    return 0;
}

💻 Fonctions avec retour

Une fonction peut renvoyer un résultat avec return.


#include 
using namespace std;

int carre(int nombre) {
    return nombre * nombre;
}

int main() {
    int resultat = carre(5);
    cout << "Carré de 5 : " << resultat << endl;
    return 0;
}

🛠 Exercice pratique

Crée une fonction qui calcule le factoriel d’un nombre (ex. : 5! = 5 * 4 * 3 * 2 * 1).


#include 
using namespace std;

int factoriel(int n) {
    int resultat = 1;
    for (int i = 1; i <= n; i++) {
        resultat *= i;
    }
    return resultat;
}

int main() {
    cout << "Factoriel de 5 : " << factoriel(5) << endl;
    return 0;
}
