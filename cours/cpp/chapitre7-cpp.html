Chapitre 7 : Tableaux et chaînes de caractères ⚙️

Stocke plusieurs valeurs ensemble dans ton programme.

🔹 Les tableaux

Un tableau est comme une rangée de boîtes numérotées, où chaque boîte contient une valeur.


#include 
using namespace std;

int main() {
    int notes[3] = {12, 15, 18};
    cout << "Première note : " << notes[0] << endl;
    return 0;
}

📌 Parcourir un tableau

Utilise une boucle pour lire ou modifier chaque élément.


#include 
using namespace std;

int main() {
    int notes[3] = {12, 15, 18};
    for (int i = 0; i < 3; i++) {
        cout << "Note " << i + 1 << " : " << notes[i] << endl;
    }
    return 0;
}

🛠 Exercice pratique

Crée un programme qui calcule la moyenne des éléments d’un tableau de notes.


#include 
using namespace std;

int main() {
    int notes[4] = {10, 15, 12, 18};
    int somme = 0;
    for (int i = 0; i < 4; i++) {
        somme += notes[i];
    }
    float moyenne = (float)somme / 4;
    cout << "Moyenne : " << moyenne << endl;
    return 0;
}
