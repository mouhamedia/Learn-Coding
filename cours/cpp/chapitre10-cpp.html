Chapitre 10 : Introduction à la POO ⚙️

Crée des objets avec des classes pour organiser ton code.

🔹 Qu’est-ce que la POO ?

La programmation orientée objet (POO) utilise des "objets" qui combinent des données (attributs) et des actions (méthodes).

📌 Définir une classe

Une classe est un modèle pour créer des objets, comme un plan pour une maison.


#include 
#include 
using namespace std;

class Voiture {
public:
    string marque;
    int vitesse;

    void afficher() {
        cout << "Voiture : " << marque << ", Vitesse : " << vitesse << endl;
    }
};

int main() {
    Voiture maVoiture;
    maVoiture.marque = "Toyota";
    maVoiture.vitesse = 120;
    maVoiture.afficher();
    return 0;
}

💻 Créer un objet

Un objet est une instance d’une classe, comme une maison construite à partir du plan.


#include 
#include 
using namespace std;

class Personne {
public:
    string nom;
    void saluer() {
        cout << "Bonjour, je suis " << nom << " !" << endl;
    }
};

int main() {
    Personne p;
    p.nom = "Alice";
    p.saluer();
    return 0;
}

🛠 Exercice pratique

Crée une classe Animal avec un attribut nom et une méthode pour afficher un message.


#include 
#include 
using namespace std;

class Animal {
public:
    string nom;
    void presenter() {
        cout << "Je suis un animal nommé " << nom << "." << endl;
    }
};

int main() {
    Animal a;
    a.nom = "Rex";
    a.presenter();
    return 0;
}
