Chapitre 2 : Premier programme C++ ⚙️

Crée et exécute ton premier programme C++.

🔹 Structure d’un programme C++

Un programme C++ commence toujours par la fonction main(). C’est comme la porte d’entrée de ton code.


#include 
using namespace std;

int main() {
    cout << "Hello, World!" << endl;
    return 0;
}

Explications :

  • #include : Permet d’utiliser cout (affichage) et cin (lecture).
  • using namespace std; : Évite d’écrire std::cout à chaque fois.
  • cout << : Affiche du texte, suivi de endl pour passer à la ligne.

📌 Afficher et lire des données

Utilise cout pour afficher et cin pour lire l’entrée de l’utilisateur.


#include 
using namespace std;

int main() {
    string nom;
    cout << "Entrez votre nom : ";
    cin >> nom;
    cout << "Salut, " << nom << " !" << endl;
    return 0;
}

Ce programme demande un nom et l’affiche avec une salutation.

💻 Compiler et exécuter

Pour faire fonctionner ton programme :

  1. Enregistre ton code dans un fichier, par exemple salut.cpp.
  2. Compile avec : g++ salut.cpp -o salut.
  3. Exécute avec : ./salut (Linux/Mac) ou salut.exe (Windows).

Tu peux aussi tester directement sur OnlineGDB.

🛠 Exercice pratique

Crée un programme qui demande l’âge de l’utilisateur et affiche un message personnalisé.


#include 
using namespace std;

int main() {
    int age;
    cout << "Quel est votre âge ? ";
    cin >> age;
    cout << "Vous avez " << age << " ans, super !" << endl;
    return 0;
}

Teste ce code sur OnlineGDB !
