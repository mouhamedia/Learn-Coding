Chapitre 8 : Pointeurs et références ⚙️

Travaille directement avec la mémoire de ton programme.

🔹 Qu’est-ce qu’un pointeur ?

Un pointeur est comme une flèche qui indique où se trouve une valeur dans la mémoire de l’ordinateur.


#include 
using namespace std;

int main() {
    int nombre = 42;
    int* pointeur = &nombre; // Stocke l'adresse de nombre
    cout << "Valeur : " << *pointeur << endl; // Affiche 42
    return 0;
}

&nombre donne l’adresse, *pointeur donne la valeur à cette adresse.

📌 Références

Une référence est un autre nom pour une variable existante.


#include 
using namespace std;

int main() {
    int age = 25;
    int& ref = age; // ref est un alias pour age
    ref = 30;
    cout << "Age : " << age << endl; // Affiche 30
    return 0;
}

💻 Passage par adresse

Les pointeurs permettent de modifier des variables dans une fonction.


#include 
using namespace std;

void augmenter(int* ptr) {
    *ptr += 10;
}

int main() {
    int valeur = 20;
    augmenter(&valeur);
    cout << "Valeur : " << valeur << endl; // Affiche 30
    return 0;
}

🛠 Exercice pratique

Crée une fonction qui échange les valeurs de deux variables en utilisant des pointeurs.


#include 
using namespace std;

void echanger(int* a, int* b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x = 5, y = 10;
    cout << "Avant : x = " << x << ", y = " << y << endl;
    echanger(&x, &y);
    cout << "Après : x = " << x << ", y = " << y << endl;
    return 0;
}
