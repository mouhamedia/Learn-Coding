Chapitre 6 : Fonctions et sous-algorithmes 🛠️

Organise ton code avec des fonctions pour le rendre clair et réutilisable !

🔹 Qu’est-ce qu’une fonction ?

Une fonction (ou sous-algorithme) est un bloc de code qui effectue une tâche spécifique et peut être réutilisé. C’est comme une machine : tu donnes des entrées, elle produit une sortie.

Analogie : Une fonction est comme une recette pour un plat spécifique que tu peux réutiliser à tout moment.

🔹 Éléments d’une fonction

  • Nom : Identifie la fonction.
  • Paramètres : Données d’entrée (facultatif).
  • Valeur de retour : Résultat renvoyé (facultatif).

💻 Exemple pratique

Voici une fonction pour calculer le carré d’un nombre :


# Pseudo-code
Fonction Carre(nombre)
    Début
        Retourner nombre * nombre
    Fin
Algorithme Principal
    Début
        n ← 5
        Afficher Carre(n)
    Fin

# Python
def carre(nombre):
    return nombre * nombre
n = 5
print(carre(n))  # Affiche : 25

📌 Points clés

  • Les fonctions rendent le code modulaire et réutilisable.
  • Les paramètres permettent de passer des données à la fonction.
  • La valeur de retour est le résultat produit par la fonction.
