Chapitre 4 : Les boucles 🔄

Répète des actions facilement avec les boucles Tant que et Pour !

🔹 Qu’est-ce qu’une boucle ?

Une boucle répète un bloc d’instructions tant qu’une condition est vraie ou pour un nombre défini d’itérations. C’est comme faire des tours de piste : tu continues jusqu’à un signal d’arrêt.

Analogie : Imagine que tu lis un livre page par page (boucle Pour) ou que tu attends qu’il fasse beau pour sortir (boucle Tant que).

🔹 Types de boucles

  • Boucle Tant que : Répète tant qu’une condition est vraie.
  • Boucle Pour : Répète pour un nombre fixe d’itérations.

💻 Exemple pratique

Voici un algorithme pour afficher les nombres de 1 à 5 :


# Pseudo-code
Algorithme AfficherNombres
    Début
        Pour i de 1 à 5 Faire
            Afficher i
        FinPour
    Fin

# Python
for i in range(1, 6):
    print(i)  # Affiche : 1, 2, 3, 4, 5

Exemple avec une boucle Tant que :


# Pseudo-code
Algorithme CompterJusquaCinq
    Début
        i ← 1
        Tant que i ≤ 5 Faire
            Afficher i
            i ← i + 1
        FinTantQue
    Fin

# Python
i = 1
while i <= 5:
    print(i)
    i += 1  # Affiche : 1, 2, 3, 4, 5

📌 Points clés

  • Utilise Pour quand le nombre d’itérations est connu.
  • Utilise Tant que quand l’arrêt dépend d’une condition.
  • Attention aux boucles infinies (oublier d’incrémenter i).
