Chapitre 8 : Recherches 🔍

Trouve des données efficacement avec les algorithmes de recherche !

🔹 Qu’est-ce qu’une recherche ?

Un algorithme de recherche trouve un élément dans un tableau. C’est comme chercher un mot dans un dictionnaire.

Analogie : Imagine chercher un livre spécifique dans une bibliothèque. Tu peux vérifier chaque étagère (linéaire) ou utiliser l’ordre alphabétique (dichotomique).

🔹 Recherche linéaire

La recherche linéaire vérifie chaque élément du tableau un par un jusqu’à trouver la valeur ou atteindre la fin.

🔹 Recherche dichotomique

La recherche dichotomique (ou binaire) divise le tableau trié en deux à chaque étape, ce qui est beaucoup plus rapide.

💻 Exemple pratique

Voici une recherche linéaire :


# Pseudo-code
Algorithme RechercheLineaire(tableau, valeur)
    Début
        Pour i de 0 à longueur(tableau)-1 Faire
            Si tableau[i] = valeur Alors
                Retourner i
            FinSi
        FinPour
        Retourner -1
    Fin

# Python
def recherche_lineaire(tableau, valeur):
    for i in range(len(tableau)):
        if tableau[i] == valeur:
            return i
    return -1
tab = [10, 20, 30, 40]
print(recherche_lineaire(tab, 30))  # Affiche : 2

📌 Points clés

  • La recherche linéaire est simple mais lente pour les grands tableaux.
  • La recherche dichotomique nécessite un tableau trié mais est plus rapide.
  • Retourne l’indice de l’élément ou -1 si non trouvé.

🚀 Pour aller plus loin

Implémente la recherche dichotomique ou modifie l’exemple pour compter combien de fois une valeur apparaît dans le tableau.
