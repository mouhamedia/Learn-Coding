Chapitre 3 : Instructions conditionnelles 🚦

Prends des décisions dans tes algorithmes avec les conditions !

🔹 Qu’est-ce qu’une instruction conditionnelle ?

Une instruction conditionnelle permet d’exécuter un bloc d’instructions seulement si une condition est vraie. C’est comme un feu de circulation : si c’est vert, avance ; sinon, arrête-toi.

Analogie : Imagine que tu décides quoi porter. S’il fait froid, tu prends un manteau ; sinon, un t-shirt.

🔹 Syntaxe et opérateurs

Les instructions conditionnelles utilisent des opérateurs comme :

  • = (égalité), > (supérieur), < (inférieur).
  • ET, OU, NON pour combiner des conditions.

Structure de base : Si condition Alors ... Sinon ... FinSi.

💻 Exemple pratique

Voici un algorithme pour vérifier si un nombre est positif :


# Pseudo-code
Algorithme EstPositif
    Entrée : nombre
    Début
        Si nombre > 0 Alors
            Afficher "Le nombre est positif."
        Sinon Si nombre = 0 Alors
            Afficher "Le nombre est zéro."
        Sinon
            Afficher "Le nombre est négatif."
        FinSi
    Fin

# Python
nombre = int(input("Entrez un nombre : "))
if nombre > 0:
    print("Le nombre est positif.")
elif nombre == 0:
    print("Le nombre est zéro.")
else:
    print("Le nombre est négatif.")

📌 Points clés

  • Les conditions permettent de choisir entre plusieurs chemins.
  • elif (Sinon Si) teste des conditions supplémentaires.
  • Les opérateurs logiques combinent plusieurs conditions.
