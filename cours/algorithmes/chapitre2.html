Chapitre 2 : Variables et affectation 📦

Apprends à stocker et manipuler des données avec les variables, la base de tout algorithme !

🔹 Qu’est-ce qu’une variable ?

Une variable est comme une boîte étiquetée où tu peux stocker une valeur (un nombre, du texte, etc.). Chaque boîte a un nom et peut contenir un type de donnée spécifique.

Analogie : Imagine une étagère avec des boîtes étiquetées "âge", "nom", etc. Tu places une valeur dans chaque boîte, et tu peux la modifier ou la récupérer plus tard.

🔹 Types de données

Les variables peuvent stocker différents types de données :

  • Entier : Un nombre sans décimale (ex. : 42).
  • Réel : Un nombre avec décimales (ex. : 3.14).
  • Chaîne : Du texte (ex. : "Bonjour").
  • Booléen : Vrai ou Faux.

🔹 L’affectation

L’affectation consiste à mettre une valeur dans une variable avec l’opérateur ← (en pseudo-code) ou = (en Python). Exemple : âge ← 25 signifie "la variable âge prend la valeur 25".

💻 Exemple pratique

Voici un algorithme qui calcule l’âge d’une personne dans 5 ans :


# Pseudo-code
Algorithme CalculAgeFutur
    Entrée : age_actuel
    Sortie : age_futur
    Début
        age_futur ← age_actuel + 5
        Afficher "Dans 5 ans, tu auras ", age_futur, " ans."
    Fin

# Python
age_actuel = 20
age_futur = age_actuel + 5
print(f"Dans 5 ans, tu auras {age_futur} ans.")  # Affiche : Dans 5 ans, tu auras 25 ans.

📌 Points clés

  • Une variable est un conteneur nommé pour une valeur.
  • Les types de données définissent ce que la variable peut contenir.
  • L’affectation met une valeur dans une variable.
