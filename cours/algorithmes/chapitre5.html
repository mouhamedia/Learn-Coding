Chapitre 5 : Tableaux et parcours 📋

Organise et manipule des listes de données avec les tableaux !

🔹 Qu’est-ce qu’un tableau ?

Un tableau est une liste ordonnée de valeurs, comme une rangée de casiers numérotés. Chaque casier a un indice (commençant à 0) et contient une valeur.

Analogie : Imagine une étagère avec des casiers étiquetés 0, 1, 2, etc., où tu ranges des nombres ou du texte.

🔹 Accéder et modifier un tableau

Tu accèdes à un élément avec son indice (ex. : tableau[0]) et tu Secondary: Tu peux modifier un élément de la même manière.

🔹 Parcourir un tableau

Pour traiter tous les éléments, on utilise une boucle Pour ou Tant que.

💻 Exemple pratique

Voici un algorithme pour calculer la somme des éléments d’un tableau :


# Pseudo-code
Algorithme SommeTableau
    Entrée : tableau
    Sortie : somme
    Début
        somme ← 0
        Pour chaque élément dans tableau Faire
            somme ← somme + élément
        FinPour
        Afficher somme
    Fin

# Python
tableau = [1, 2, 3, 4]
somme = 0
for element in tableau:
    somme += element
print(somme)  # Affiche : 10

📌 Points clés

  • Un tableau est une collection ordonnée accessible par indices (0, 1, 2...).
  • Utilise une boucle pour parcourir tous les éléments.
  • Les tableaux sont utiles pour stocker des listes de données.

🚀 Pour aller plus loin

Modifie l’exemple pour trouver le plus grand élément du tableau ou pour afficher les éléments en ordre inverse.
