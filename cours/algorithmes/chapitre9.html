Chapitre 9 : Algorithmes classiques 🌟

Découvre des algorithmes célèbres comme le PGCD et Fibonacci !

🔹 Qu’est-ce qu’un algorithme classique ?

Les algorithmes classiques sont des solutions bien connues à des problèmes courants, comme calculer le PGCD ou générer la suite de Fibonacci.

Analogie : Ce sont comme des outils éprouvés dans une boîte à outils, utilisés dans de nombreux programmes.

🔹 PGCD (Euclide)

L’algorithme d’Euclide calcule le plus grand commun diviseur de deux nombres en utilisant des divisions successives.

🔹 Suite de Fibonacci

La suite de Fibonacci génère des nombres où chaque terme est la somme des deux précédents (0, 1, 1, 2, 3, 5, 8...).

💻 Exemple pratique

Voici l’algorithme du PGCD d’Euclide :


# Pseudo-code
Algorithme PGCD(a, b)
    Début
        Tant que b ≠ 0 Faire
            temp ← b
            b ← a mod b
            a ← temp
        FinTantQue
        Retourner a
    Fin

# Python
def pgcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a
print(pgcd(48, 18))  # Affiche : 6

Exemple pour Fibonacci :


# Pseudo-code
Algorithme Fibonacci(n)
    Début
        Si n ≤ 1 Alors
            Retourner n
        Sinon
            a ← 0
            b ← 1
            Pour i de 2 à n Faire
                c ← a + b
                a ← b
                b ← c
            FinPour
            Retourner b
        FinSi
    Fin

# Python
def fibonacci(n):
    if n <= 1:
        return n
    a, b = 0, 1
    for _ in range(2, n+1):
        a, b = b, a + b
    return b
print(fibonacci(6))  # Affiche : 8

📌 Points clés

  • Le PGCD d’Euclide est efficace grâce aux divisions successives.
  • La suite de Fibonacci peut être calculée itérativement pour plus d’efficacité.
  • Ces algorithmes sont utilisés dans de nombreux domaines (maths, cryptographie, etc.).
